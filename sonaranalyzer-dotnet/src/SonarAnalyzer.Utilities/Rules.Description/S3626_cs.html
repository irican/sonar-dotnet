跳转语句，例如 return，yield break，goto 以及 continue 允许你改变程序执行的默认流（default flow），但是令控制流跳回原方向的跳转语句则是没有意义的代码。

不合规的代码示例

void Foo()
{
  goto A; // 不合规
  A:
  while (condition1)
  {
    if (condition2)
    {
      continue; // 不合规
    }
    else
    {
      DoTheThing();
    }
  }
  return; // 不合规，这是一个void方法
}

合规的代码示例

void Foo()
{
  while (condition1)
  {
    if (!condition2)
    {
      DoTheThing();
    }
  }
}
