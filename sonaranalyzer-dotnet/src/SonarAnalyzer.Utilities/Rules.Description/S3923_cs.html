在 switch 的所有分支或在 if 的链语句中都做相同的实现是一个错误。要么是复制粘贴的错误更改了原本不同的操作，要么就根本不应该有这条 switch／if 链语句。

不合规的代码示例

if (b == 0)  // 不合规
{
    DoTheThing();
}
else
{
    DoTheThing();
}

int b = a > 12 ? 4 : 4;  // 不合规

switch (i) // 不合规
{
    case 1:
        DoSomething();
        break;
    case 2:
        DoSomething();
        break;
    case 3:
        DoSomething();
        break;
    default:
        DoSomething();
}

例外

此规则对没有 else 的 if 链语句以及没有 default 子句的 switch 语句不适用。

if (b == 0)    //没有问题，这种写法可能是有意让代码变得易读
{
    DoSomething();
}
else if (b == 1)
{
    DoSomething();
}
