多余的圆括号是无用的，应该将其删除。

不合规的代码示例

[MyAttribute()] //不合规
class MyClass
{
  public int MyProperty { get; set; }
  public static MyClass CreateNew(int propertyValue)
  {
    return new MyClass() //不合规
    {
      MyProperty = propertyValue
    };
  }
}

合规的解决方案

[MyAttribute]
class MyClass
{
  public int MyProperty { get; set; }
  public static MyClass CreateNew(int propertyValue)
  {
    return new MyClass
    {
      MyProperty = propertyValue
    };
  }
}
