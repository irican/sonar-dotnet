对于可以跨应用程序域边界访问的对象，想要获取该对象的锁的线程存在被其他应用程序域中另一线程阻塞的风险。可以跨应用程序域边界访问的对象被称为具有弱特征（weak identity），具有弱特征的类型有：

  • MarshalByRefObject
  • ExecutionEngineException
  • OutOfMemoryException
  • StackOverflowException
  • String
  • MemberInfo
  • ParameterInfo
  • Thread

不合规的代码示例

using System;
using System.Threading;

namespace MyLibrary
{
  class Foo
  {
    string myString = "foo";

    void Bar()
    {
      lock(myString) { } // 不合规
    }
  }
}

合规的解决方案

using System;
using System.Threading;

namespace MyLibrary
{
  class Foo
  {
    string myString = "foo";
    private readonly Object thisLock = new Object();

    void Bar()
    {
      lock(thisLock) { } // 合规
    }
  }
}
