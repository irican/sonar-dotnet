可为null的值类型可以有实际值，也可以为 null。该类型的值可以用 Value 属性访问，但 .Value 会在它的值为 null 时抛出一个 InvalidOperationException 异常。为避免这个异常，这种类型在用 .Value 访问前应进行检查。

不合规的代码示例

int? nullable = null;
...
UseValue(nullable.Value); // 不合规

合规的解决方案

int? nullable = null;
...
if (nullable.HasValue)
{
  UseValue(nullable.Value);
}

或者

int? nullable = null;
...
if (nullable != null)
{
  UseValue(nullable.Value);
}
