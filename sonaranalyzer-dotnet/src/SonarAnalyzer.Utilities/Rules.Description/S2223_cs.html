static 字段既不是常量也不是只读的，这不满足线程安全（thread-safe）。要想正确地在不同的线程里访问这些字段就需要实现带有 lock 的同步机制。不正确的同步操作可能会产生意外的结果，因此公共可见的静态字段最适合存储有多个使用者使用的固定不变的数据。为实现这个目的，这些字段应该标为 readonly 或转换为常量。

不合规的代码示例

public class Math
{
  public static double Pi = 3.14;  // 不合规
}

或者

public class Shape
{
  public static Shape Empty = new EmptyShape();  // 不合规

  private class EmptyShape : Shape
  {
  }
}

合规的解决方案

public class Math
{
  public const double Pi = 3.14;
}

或者

public class Shape
{
  public static readonly Shape Empty = new EmptyShape();

  private class EmptyShape : Shape
  {
  }
}
