IntPtr 和 UIntPtr 类型用于访问非托管内存，通常是为了使用C或C++库。这样的指针如果不设为 private，internal 或 readonly 类型加以保护，就可能会产生允许访问任意位置的漏洞。

不合规的代码示例

using System;

namespace MyLibrary
{
  public class MyClass
  {
    public IntPtr myPointer;  // 不合规
    protected UIntPtr myOtherPointer; // 不合规
  }
}

合规的解决方案

using System;

namespace MyLibrary
{
  public class MyClass
  {
    private IntPtr myPointer;
    protected readonly UIntPtr myOtherPointer;
  }
}
