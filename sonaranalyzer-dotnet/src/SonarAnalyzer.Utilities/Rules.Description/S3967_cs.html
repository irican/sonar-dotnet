交错数组是以数组为元素的数组。建议用它代替多维数组，因为它的元素数组可以有不同的大小，这样避免了空间的浪费。

不合规的代码示例

int [,] myArray =  // 不合规
    {
        {1,2,3,4},
        {5,6,7,0},
        {8,0,0,0},
        {9,0,0,0}
    };
// ……
myArray[1,1] = 0;

合规的解决方案

int[][] myArray =
    {
        new int[] {1,2,3,4},
        new int[] {5,6,7},
        new int[] {8},
        new int[] {9}
    };
// ……
myArray[1][1] = 0;
