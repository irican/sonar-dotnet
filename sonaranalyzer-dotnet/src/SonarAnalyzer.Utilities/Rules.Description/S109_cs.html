魔术数（magic number）指没有说明来源（come out of nowhere），直接出现在语句中的具体数字。魔术数的使用很常见，例如，在限定一条循环语句的循环次数时，或是测试一个属性的值时，等等。

当你在写代码时，魔术数对你也许是简单易懂的，但在调试时对他人可就不是了。

所以必须在使用魔术数之前将它赋给一个有名称的变量。

－1，0，1不视作魔术数。

不合规的代码示例

public static void DoSomething()
{
    for(int i = 0; i < 4; i++)  // 不合规，4是一个魔术数
    {
        ...
    }
}

合规的解决方案

private const int NUMBER_OF_CYCLES = 4;

public static void DoSomething()
{
    for(int i = 0; i < NUMBER_OF_CYCLES ; i++)  //合规
    {
        ...
    }
}

例外

出现在 GetHashCode 方法与变量／字段声明中的魔术数不会引发问题。
