从C# 7开始，可以给数字常量添加下划线（'_'）以增强代码的可读性。这样添加的下划线没有语义上的含义，只是为了让维护者更方便地理解代码。

小数点左边有多少位时才使用此规则要根据基数来判断。

Base Minimum digits
binary 9
decimal 6
hexadecimal 9

此规则只检查是否使用了下划线，对它们的间隔不做检查。

注意此规则不适用于 C# version 低于 7的项目。

不合规的代码示例

int i = 10000000;  // 不合规；这是一千万还是一亿呢？
int  j = 0b01101001010011011110010101011110;  // 不合规
long l = 0x7fffffffffffffffL;  // 不合规

合规的解决方案

int i = 10_000_000;
int  j = 0b01101001_01001101_11100101_01011110;
long l = 0x7fff_ffff_ffff_ffffL;
