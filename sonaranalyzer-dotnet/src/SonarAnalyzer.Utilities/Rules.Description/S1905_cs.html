不必要的强制类型转换表达式会让代码难以阅读和理解

不合规的代码示例

public int Example(int i)
{
  return (int) (i + 42); // 不合规
}
public IEnumerable<int> ExampleCollection(IEnumerable<int> coll)
{
  return coll.Reverse().OfType<int>(); // 不合规
}

合规的解决方案

public int Example(int i)
{
  return i + 42;
}
public IEnumerable<int> ExampleCollection(IEnumerable<int> coll)
{
  return coll.Reverse();
}
