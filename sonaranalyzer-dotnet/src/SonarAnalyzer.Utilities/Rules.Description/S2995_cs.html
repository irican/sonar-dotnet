大多数情况下使用 Object.ReferenceEquals 来比较两个值类型的引用不会返回想要的结果，因为这种类型是通过值传递的，而不是通过引用传递的。

不合规的代码示例

public class MyClass
{
  private MyStruct myStruct;

  public void DoSomething(MyStruct s1) {
    int a = 1;
    int b = 1;

    if (Object.ReferenceEquals(myStruct, s1))  // 不合规，此处不可能为true
    {
      // ……
    }
    else if (Object.ReferenceEquals(a,b)) // 不合规
    {
      // ……
    }
  }
}
