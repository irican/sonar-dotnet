用 Pure 特性标记某个方法意在指明该方法不会做任何可见的修改。因此该方法应返回一个结果，否则调用此方法等同于无操作。所以，用 Pure 标记 void 型方法要么是个错误，要么说明该方法不做任何有意义的操作。

不合规的代码示例

class Person
{
  private int age;
  [Pure] // 不合规，该方法可能会进行可见的状态修改
  void ConfigureAge(int age)
  {
    ...
    this.age = age;
  }
  ...
}

合规的解决方案

class Person
{
  private int age;

  void ConfigureAge(int age)
  {
    ...
    this.age = age;
  }
  ...
}
