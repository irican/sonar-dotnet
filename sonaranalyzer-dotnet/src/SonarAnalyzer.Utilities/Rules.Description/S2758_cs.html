当三元运算符的第二和第三个操作数相同时，无论条件如何该运算符始终返回相同的结果。要么该运算符是无效的，要么就是编码时出现了错误。

不合规的代码示例

public bool CanVote(Person person)
{
  return person.GetAge() > 18 ? true : true; // 不合规，这是开发者的真实意图吗？
}

合规的解决方案

public bool CanVote(Person person)
{
  return person.GetAge() > 18 ? true : false;
  // 更好的写法：
  // return person.GetAge() > 18;
}

弃用

此规则已被弃用，请换用 {rule:csharpsquid:S3923}
