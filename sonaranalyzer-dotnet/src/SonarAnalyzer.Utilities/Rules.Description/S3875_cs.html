在用 == 比较对象时进行的是引用比较。也就是说，当且仅当受比较的对象是同一个对象实例时，才会返回 true。除此之外，如果想执行其他的操作而重载该运算符会让使用者不可避免地引入bug。而另一方面，如果没有其他的操作，那么就没有重载该运算符的意义，因为默认情况下 == 就是在进行比较。

例外

  • 重载了 operator + 或 operator - 的类被此规则忽略。
  • 实现 IComparable<T> 或 IEquatable<T> 的类很可能表现为值类型的对象，因此被此规则 忽略。
