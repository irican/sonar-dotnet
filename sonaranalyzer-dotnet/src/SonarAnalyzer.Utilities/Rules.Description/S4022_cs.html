默认情况下，enum 的存储类型是 Int32。在大多数情况下，没有必要改变这种情况。特别是，使用较小的数据类型(例如 Byte) 不会获得任何性能提升，而且可能会限制未来的使用。

不合规的代码示例

using System;

namespace MyLibrary
{
    public enum Visibility : sbyte // 不合规
    {
        Visible = 0,
        Invisible = 1,
    }
}

合规的解决方案

using System;

namespace MyLibrary
{
    public enum Visibility
    {
        Visible = 0,
        Invisible = 1,
    }
}
