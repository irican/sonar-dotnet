当异常发生时，忽略它们通常不是个好想法。相反，你最好能正确地处理它们，或是至少将它们记录下来。

此规则只报告那些捕获一般（generic）Exception 的空catch子句。

不合规的代码示例

string text = "";
try
{
    text = File.ReadAllText(fileName);
}
catch (Exception exc) // 不合规
{
}

合规的解决方案

string text = "";
try
{
    text = File.ReadAllText(fileName);
}
catch (Exception exc)
{
    logger.Log(exc);
}

例外

包含了注释的块语句不被认为是空的。
