如果你对一个 Type 变量调用 GetType()，返回值总会为 typeof(System.Type)。因此这样的调用没有意义。将一个类型参数传递给 IsInstanceOfType 同样如此。这两种操作的结果都是完全可以预见到的。

不合规的代码示例

var type = typeof(int);
var ttype = type.GetType(); //不合规，结果必然是typeof(System.Type)

var s = "abc";

if (s.GetType().IsInstanceOfType(typeof(string))) //不合规，false
{ /* …… */ }

合规的解决方案

var s = "abc";

if (s.GetType().IsInstanceOfType("ssss"))
{ /* …… */ }
