重写方法会自动继承 params 行为。为了增强可读性，应该在重写方法中也显式地写明此标识符。

不合规的代码示例

class Base
{
  public virtual void Method(params int[] numbers)
  {
    ...
  }
}
class Derived : Base
{
  public override void Method(int[] numbers) // 不合规，缺失了params
  {
    ...
  }
}

合规的解决方案

class Base
{
  public virtual void Method(params int[] numbers)
  {
    ...
  }
}
class Derived : Base
{
  public override void Method(params int[] numbers)
  {
    ...
  }
}
