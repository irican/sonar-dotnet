为了保证代码的简洁，应使用最简单的条件语法，即：

  • 使用 ?? 作为“如果不为空则赋值”运算符，
  • 对单个变量赋值时使用三元运算符 ?: ，并且
  • 已知左操作数为 null 或者绝对不为 null 时移除 ??。

不合规的代码示例

object a = null, b = null, x;

if (a != null) // 不合规，完全是多余的
{
  x = a;
}
else
{
  x = b;
}

x = a != null ? a : b; // 不合规，仍可进一步简化

x = (a == null) ? new object() : a; // 不合规

if (condition) // 不合规
{
  x = a;
}
else
{
  x = b;
}

var y = null ?? new object(); // 不合规

合规的解决方案

object x;

x = a ?? b;
x = a ?? b;
x = a ?? new object();
x = condition ? a : b;
var y = new object();
