使用throw抛出 Exception，SystemException， ApplicationException，IndexOutOfRangeException，NullReferenceException，OutOfMemoryException 以及 ExecutionEngineException 这样的一般异常，可让调用的方法不必处理真正的、系统产生的异常，而这与应用产生的错误是不同的。

不合规的代码示例

public void DoSomething(object obj)
{
  if (obj == null)
  {
    throw new NullReferenceException("obj");  // 不合规
  }
  // ……
}

合规的解决方案

public void DoSomething(object obj)
{
  if (obj == null)
  {
    throw new ArgumentNullException("obj");
  }
  // ……
}

请参阅

  • MITRE, CWE-397 - Declaration of Throws for Generic Exception
  • CERT, ERR07-J. - Do not throw RuntimeException, Exception, or Throwable
