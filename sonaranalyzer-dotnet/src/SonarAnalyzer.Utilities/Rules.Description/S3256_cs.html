用 string.Equals 来确定字符串是否为空会明显地比用 string.IsNullOrEmpty() 或直接检查 string.Length == 0 要慢得多。string.IsNullOrEmpty() 既明确又简洁，因此更多用于费时的、容易出错的手动null值检查与空（emptiness）检查。

不合规的代码示例

"".Equals(name); // 不合规
!name.Equals(""); // 不合规
name.Equals(string.Empty); // 不合规

合规的解决方案

name != null && name.Length > 0 // 合规但更容易出错
!string.IsNullOrEmpty(name)
string.IsNullOrEmpty(name)
