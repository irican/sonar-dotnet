如果已要求调用者给参数提供一个值，那么再给参数提供默认值就是没有意义的。因此 [DefaultParameterValue] 应总是和 [Optional] 一起使用。

不合规的代码示例

public void MyMethod([DefaultParameterValue(5)] int j) //不合规，起不到实际作用
{
  Console.WriteLine(j);
}

合规的解决方案

public void MyMethod(int j = 5)
{
  Console.WriteLine(j);
}

或者

public void MyMethod([DefaultParameterValue(5)][Optional] int j)
{
  Console.WriteLine(j);
}
