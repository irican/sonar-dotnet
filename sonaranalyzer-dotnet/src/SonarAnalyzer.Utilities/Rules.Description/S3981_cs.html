集合的大小与数组的长度总是大于等于零的。所以检验它们是否大于等于零是没有意义的，因为结果总为 true。类似地，检验是否小于零也总会返回 false。也许这种检查的意图是判断集合与数组是否非空。

不合规的代码示例

if(collection.Count >= 0){...}

if(enumerable.Count() < 0){...}

if(array.Length >= 0){...}

bool result = array.Length >=0;

合规的解决方案

if (list.Any()) { ... }

if (list.Count > 0) { ... }

if (array.Length >= 42) { ... }
