NotImplementedException 常用于标记一类特定的方法：这些方法在自身被实现后总体功能才能完成，然而开发人员希望稍后再实现这些它们。这与 NotSupportedException 相反，该异常由另一类方法引发，这些方法被基类或接口所需要，但它们却不适用于当前的类型。

抛出 NotImplementedException 异常时此规则会引发一个问题。

不合规的代码示例

void doTheThing()
{
    throw new NotImplementedException();
}

例外

派生自 NotImplementedException 的异常被此规则忽略。
