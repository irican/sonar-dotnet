从未分配默认值的字段和自动属性（auto-property）可能是无用的代码，更有可能是个错误。

不合规的代码示例

class MyClass
{
  private int field; // 不合规，不应该将其初始化吗？这样会让该字段总是取默认值0
  private int Property { get; set; }  // 不合规
  public void Print()
  {
    Console.WriteLine(field); //总是输出0
    Console.WriteLine(Property); //总是输出0
  }
}

合规的解决方案

class MyClass
{
  private int field = 1;
  private int Property { get; set; } = 42;
  public void Print()
  {
    field++;
    Console.WriteLine(field);
    Console.WriteLine(Property);
  }
}
