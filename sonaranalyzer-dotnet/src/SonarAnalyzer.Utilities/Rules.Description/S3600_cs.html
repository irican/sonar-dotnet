向重写方法添加 params 是无效的。编译器可以通过，但方法的调用者是无法从这个添加的修饰符中获益的。

不合规的代码示例

class Base
{
  public virtual void Method(int[] numbers)
  {
    ...
  }
}
class Derived : Base
{
  public override void Method(params int[] numbers) // 不合规，无法以params参数的使用方法调用该方法
  {
    ...
  }
}

合规的解决方案

class Base
{
  public virtual void Method(int[] numbers)
  {
    ...
  }
}
class Derived : Base
{
  public override void Method(int[] numbers)
  {
    ...
  }
}
