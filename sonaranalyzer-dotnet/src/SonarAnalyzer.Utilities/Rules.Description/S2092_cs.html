“secure”特性防止cookie在明文连接上被传输，如HTTP，在该连接中cookie很容易被窃听。因此，带有安全特性的cookies只能在加密的HTTPS连接上传输。

推荐的安全编码习惯

  • 在 HttpCookie 对象上将 Secure 字段设为 true。

不合规的代码示例

HttpCookie myCookie = new HttpCookie("UserSettings");
myCookie.Secure = false; // 不合规，显式地设为了false
...
Response.Cookies.Add(myCookie);

HttpCookie myCookie = new HttpCookie("UserSettings"); // 不合规；使用了‘Secure’的默认值（即false）
...
Response.Cookies.Add(myCookie);

合规的解决方案

HttpCookie myCookie = new HttpCookie("UserSettings");
myCookie.Secure = true; // 合规
...
Response.Cookies.Add(myCookie);
