有一些有效的情况可以将变量多次传递到相同的方法调用中，但是这样做通常是错误的，而且还需要为其中一个参数指定其他内容。

不合规的代码示例

if (Compare(point.X, point.X) != 0) // 不合规
{
  //...
}

if (DoSomething(GetNextValue(), GetNextValue()))  // 不合规
{
  // ...
}

合规的解决方案

if (Compare(point.X, point.Y) != 0)
{
  //...
}

var v1 = GetNextValue();
var v2 = GetNextValue();
if (DoSomething(v1, v2))
{
  // ...
}
