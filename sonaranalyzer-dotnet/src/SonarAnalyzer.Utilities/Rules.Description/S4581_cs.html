当使用语法new Guid() (即无参数实例化)时，必须满足以下三个条件之一：

  1. 空的GUID，在这种情况下Guid.Empty 更清晰。
  2. 一个随机生成的GUID，在这种情况下应该使用 Guid.NewGuid()。
  3. 具有特定初始化的新GUID，在这种情况下缺少初始化参数。

当发现Guid struct的无参数实例化时，此规则会引发一个问题。

不合规的代码示例

public void Foo()
{
    var g = new Guid(); // 不合规 - 目的是什么？
}

合规的解决方案

public void Foo(byte[] bytes)
{
    var g1 = Guid.Empty;
    var g2 = Guid.NewGuid();
    var g3 = new Guid(bytes);
}
