由于从编译过的应用程序中提取字符串很容易，所以不应该将凭证硬编码。这样做的话，它们最终几乎肯定会落入攻击者手中。分布式的应用程序尤其如此。

证书不该在代码里，而是应该保存在一个受到强力保护的加密配置文件或者数据库中。

不合规的代码示例

string username = "admin";
string password = "Password123"; // 不合规
string usernamePassword  = "user=admin&password=Password123"; // 不合规
string usernamePassword2 = "user=admin&" + "password=" + password; // 不合规

合规的解决方案

string username = "admin";
string password = GetEncryptedPassword();
string usernamePassword = string.Format("user={0}&password={1}", GetEncryptedUsername(), GetEncryptedPassword());
