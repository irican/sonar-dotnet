方法解析的规则很复杂，可能不是所有的编程人员都能正确理解。用可选参数值对方法重载可能会让理解变得更加困难。

当没有可选参数的方法重载隐藏了使用默认参数值的方法重载时，此规则会引发一个问题。

不合规的代码示例

public class MyClass
{
  void Print(string[] messages) {...}
  void Print(string[] messages, string delimiter = "\n") {...} // 不合规，默认参数值被重载方法隐藏
}

// ……
MyClass myClass = new MyClass();

myClass.Print(new string[3] {"yes", "no", "maybe"});  // 会调用哪个版本的Print？
