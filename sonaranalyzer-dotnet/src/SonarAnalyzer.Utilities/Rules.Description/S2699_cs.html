没有断言（assert）的测试用例只能保证不抛出异常，除了基本的可运行性外，它无法保证能测试代码的行为。

此规则会在测试中的以下任何框架里未找到断言时引发异常：

  • MSTest
  • NUnit
  • XUnit
  • FluentAssertions

不合规的代码示例

[TestMethod]
public void MyMethod_WhenSomething_ExpectsSomething()
{
    var myClass = new Class();
    var result = myClass.GetFoo();
}

合规的解决方案

[TestMethod]
public void MyMethod_WhenSomething_ExpectsSomething()
{
    var myClass = new Class();
    var result = myClass.GetFoo();
    Assert.IsTrue(result);
}
