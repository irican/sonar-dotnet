当程序集使用Windows窗体（Forms）(来自System.Windows.Forms名称空间的类和接口)时，它的入口点应标明为STAThreadAttribute，以指示线程模型应为"单线程单元（STA），这是Windows窗体支持的唯一模型。

当使用Windows窗体的程序集的入口点(static void Main 方法)有标记为STA时，该规则会引发一个问题。

不合规的代码示例

using System;
using System.Windows.Forms;

namespace MyLibrary
{
    public class MyForm: Form
    {
        public MyForm()
        {
            this.Text = "Hello World!";
        }

        public static void Main()  // 不合规
        {
            var form = new MyForm();
            Application.Run(form);
        }
    }
}

合规的解决方案

using System;
using System.Windows.Forms;

namespace MyLibrary
{
    public class MyForm: Form
    {
        public MyForm()
        {
            this.Text = "Hello World!";
        }

        [STAThread]
        public static void Main()
        {
            var form = new MyForm();
            Application.Run(form);
        }
    }
}
