不重要的属性，即没有其他的逻辑，仅仅set与get一个字段（a backing field）的属性，应将其转换为自动实现的（auto-implemented）的属性，这样能得到更为简洁易懂的代码。

不合规的代码示例

public class Car
{
  private string _make;
  public string Make // 不合规
  {
    get { return _make; }
    set { _make = value; }
  }
}

合规的解决方案

public class Car
{
  public string Make { get; set; }
}
