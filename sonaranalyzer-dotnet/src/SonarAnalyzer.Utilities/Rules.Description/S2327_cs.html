当多个相邻的 try 语句都有相同的 catch 和／或 finally 语句块内容时，应该将它们写在一起，合并这些 catch/finally 逻辑以得到更简单更易懂的代码。注意，即便是这些 try 语句块之间有间隔的其他代码，此规则也同样适用。

不合规的代码示例

try
{
  DoTheFirstThing(a, b);
}
catch (InvalidOperationException ex)
{
  HandleException(ex);
}

DoSomeOtherStuff();

try  // 不合规；这里的catch和上一个内容相同
{
  DoTheSecondThing();
}
catch (InvalidOperationException ex)
{
  HandleException(ex);
}

try  // 合规；catch对异常的处理与之前不同
{
  DoTheThirdThing(a);
}
catch (InvalidOperationException ex)
{
  LogAndDie(ex);
}

合规的解决方案

try
{
  DoTheFirstThing(a, b);
  DoSomeOtherStuff();
  DoTheSecondThing();
}
catch (InvalidOperationException ex)
{
  HandleException(ex);
}

try  // 合规；catch对异常的处理与之前不同
{
  DoTheThirdThing(a);
}
catch (InvalidOperationException ex)
{
  LogAndDie(ex);
}
