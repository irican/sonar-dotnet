调用方法时，相同的形参名和实参名能使代码更清晰易懂。但如果名称相同，参数顺序却不匹配，这说明传入参数时的顺序可能有错，最后也很可能会得到一个意外的结果。

不合规的代码示例

public double Divide(int divisor, int dividend)
{
  return divisor/dividend;
}

public void DoTheThing()
{
  int divisor = 15;
  int dividend = 5;

  double result = Divide(dividend, divisor);  // 不合规；操作是成功的，但结果并不是想要的
  //……
}

合规的解决方案

public double Divide(int divisor, int dividend)
{
  return divisor/dividend;
}

public void DoTheThing()
{
  int divisor = 15;
  int dividend = 5;

  double result = Divide(divisor, dividend);
  //……
}
