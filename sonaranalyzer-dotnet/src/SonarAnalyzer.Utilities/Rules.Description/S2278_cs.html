根据美国国家标准及技术研究所（NIST）的声明，数据加密标准不再被认为是安全的（DES）：

DES在1977被联邦机构采用，用于保护敏感而又尚未加密的信息。但由于之后在保护联邦政府信息时其安全性达不到要求，因此又被撤销。

联邦机构被建议使用高级加密标准（AES），该算法于2001年被批准为FIPS 197，是一种更快更强的算法。

类似地，RC2也应该被弃用。

不合规的代码示例

using (var tripleDES = new TripleDESCryptoServiceProvider()) //不合规
{
  //……
}

合规的解决方案

using (var aes = new AesCryptoServiceProvider())
{
  //……
}
