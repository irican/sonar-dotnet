Dispose 作为方法名时应仅用于实现 IDisposable.Dispose 的时候，这样可以避免混淆。

为其他的目的新建一个 Dispose 方法是很有诱惑力的想法，但这样做会让人困惑，并很可能在生产中产生问题。

不合规的代码示例

public class GarbageDisposal
{
  private int Dispose()  // 不合规
  {
    // ...
  }
}

合规的解决方案

public class GarbageDisposal : IDisposable
{
  public void Dispose()
  {
    // ……
  }
}

或者

public class GarbageDisposal
{
  private int Grind()
  {
    // ……
  }
}

例外

如果有方法名为 Dispose，并且在 IDisposable.Dispose 的实现中被调用，则此规则不会报告此方法。

public class GarbageDisposal  :  IDisposable
{
  protected virtual void Dispose(bool disposing)
  {
    //……
  }
  public void Dispose()
  {
    Dispose(true);
    GC.SuppressFinalize(this);
  }
}
