字符串表示的URI或URL易出现解析和编码错误，这些错误可能会导致漏洞的产生。System.Uri 类是一种安全的替代方案，应予以优先考虑。

当某个方法返回 string 类型，且其名称包含“Uri”，“Urn”，“Url”，或以“uri”，“urn”与“url”开头时，此规则引发一个问题。

不合规的代码示例

using System;

namespace MyLibrary
{
   public class MyClass
   {
      public string GetParentUri() // 不合规
      {
         return "http://www.mysite.com";
      }
   }
}

合规的解决方案

using System;

namespace MyLibrary
{
   public class MyClass
   {

      public Uri GetParentUri()
      {
         return new URI("http://www.mysite.com");
      }
   }
}
