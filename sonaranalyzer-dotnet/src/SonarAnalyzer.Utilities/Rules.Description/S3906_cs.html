委托事件处理器（Delegate event handlers）（即用作事件类型的委托）的签名应该非常具体：

  • 返回值类型为 void.
  • 第一个参数是 System.Object 类并且命名为‘sender’
  • 第二个参数是 System.EventArgs 类（或任何它的派生类）并且命名为‘e’

当有某个 delegate 的声明不符合以上签名时，此规则会引发一个问题。

不合规的代码示例

public delegate void AlarmEventHandler(object s);

public class Foo
{
    public event AlarmEventHandler AlarmEvent; // 不合规
}

合规的解决方案

public delegate void AlarmEventHandler(object sender, AlarmEventArgs e);

public class Foo
{
    public event AlarmEventHandler AlarmEvent; // 合规
}
