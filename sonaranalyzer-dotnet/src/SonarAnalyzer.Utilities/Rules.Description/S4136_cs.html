为确保清晰，所有对于同一方法的重载应该放在一起，这样用户和维护者就可以很快理解所有可能的选项。

不合规的代码示例

interface IMyInterface
{
  int DoTheThing(); // 不合规 - 重载方法声明没有放在一起
  string DoTheOtherThing();
  int DoTheThing(string s);
}

合规的解决方案

interface IMyInterface
{
  int DoTheThing();
  int DoTheThing(string s);
  string DoTheOtherThing();
}

例外

将相同访问等级的方法放在一起也是编程的常例，所以访问修饰符不同的重载方法不报告问题。

例如

class MyClass
{
  private void DoTheThing(string s) // 没问题 - 这是私有（private）方法，另外一个是公共的（public）
  {
    // ...
  }

  private string DoTheOtherThing(string s)
  {
    // ...
  }

  public void DoTheThing()
  {
    // ...
  }
}
