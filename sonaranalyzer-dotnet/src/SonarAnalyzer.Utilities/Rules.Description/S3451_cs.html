将 [DefaultValue] 与 [Optional] 一起使用只能产生 [Optional] 的效果。因为实际上 [DefaultValue] 什么都没做，它只表明了某个值的使用意图。使用 [DefaultValue] 的行为很可能是混淆了它与 [DefaultParameterValue]。

不合规的代码示例

class MyClass
{
    public void DoStuff([Optional][DefaultValue(4)]int i, int j = 5)  // 不合规
    {
        Console.WriteLine(i);
    }

    public static void Main()
    {
        new MyClass().DoStuff(); // 输出0
    }
}

合规的解决方案

class MyClass
{
    public void DoStuff([Optional][DefaultParameterValue(4)]int i, int j = 5)
    {
        Console.WriteLine(i);
    }

    public static void Main()
    {
        new MyClass().DoStuff(); //输出4
    }
}
