在一个LINQ中链接多个 OrderBy 调用是没有意义的，因为只有最后一个会在结果中反映出来。这是因为每一次调用都会完全将列表重新排序。所以，多次调用 OrderBy 也是一个性能问题，因为所有的排序都会被执行，但只有最后一次的结果会被保留。

所以，应在第一次调用后换用 ThenBy。

不合规的代码示例

var x = personList
  .OrderBy(person => person.Age)
  .OrderBy(person => person.Name)  // 不合规
  .ToList();  // x是依Name排序的结果，而不是一个子排序

合规的解决方案

var x = personList
  .OrderBy(person => person.Age)
  .ThenBy(person => person.Name)
  .ToList();
