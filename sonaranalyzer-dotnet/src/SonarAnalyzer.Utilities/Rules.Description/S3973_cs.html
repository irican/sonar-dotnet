在没有花括号的情况下，条件之后紧跟的一行就是根据条件执行的行。不论是出于使用约定，还是良好编程习惯的要求，这些行都是要缩进的。而如果花括号和行缩进都没有，那么原程序员的意图就是完全不清楚的，或者其意图并不是执行什么。此外，这样的代码很可能会让维护人员感到困惑。

不合规的代码示例

if (condition)  // 不合规
DoTheThing();

DoTheOtherThing();
SomethingElseEntirely();

Foo();

合规的解决方案

if (condition)
  DoTheThing();

DoTheOtherThing();
SomethingElseEntirely();

Foo();
