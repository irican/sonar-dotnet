字符串表示的URI或URL易出现解析和编码错误，这些错误可能会导致漏洞的产生。System.Uri 类是一种安全的替代方案，应予以优先考虑。

当某个属性是字符串型，并且其名称包含“uri”，“Uri”，“urn”，“Urn”，“url”或“Url”时，此规则会引发一个问题。

不合规的代码示例

using System;

namespace MyLibrary
{
   public class MyClass
   {
      string myUri;

      public string MyUri // 不合规
      {
         get { return myURI; }
         set { myUri = value; }
      }
   }
}

合规的解决方案

using System;

namespace MyLibrary
{
   public class MyClass
   {
      Uri myUri;

      public Uri MyUri
      {
         get { return myURI; }
         set { myUri = value; }
      }
   }
}
