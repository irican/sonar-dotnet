跳转到default子句的空 case 子句是无用的，因为无论该 case 是否存在，default 子句都会被调用。这种 case 只会让代码变得混乱，应该直接删除。

不合规的代码示例

switch(ch)
{
  case 'a' :
    HandleA();
    break;
  case 'b' :
    HandleB();
    break;
  case 'c' :  // 不合规
  default:
    HandleTheRest();
    break;
}

合规的解决方案

switch(ch)
{
  case 'a' :
    HandleA();
    break;
  case 'b' :
    HandleB();
    break;
  default:
    HandleTheRest();
    break;
}
