如果未将泛型的类型参数（generic type parameter）限制为引用类型的，那么类似 struct 这样的值类型参数也可以被传递。在这种情况下，将值类型的类型参数与 null 比较的结果总是为false，因为 struct 可以为空（empty），但不可能为 null。如果确实是想使用值类型的参数，那么比较时应使用 default()；如果不是那么就应增加限制，防止值类型参数的传递。

不合规的代码示例

private bool IsDefault<T>(T value)
{
  if (value == null) // 不合规
  {
    // ……
  }
  // ……
}

合规的解决方案

private bool IsDefault<T>(T value)
{
  if(object.Equals(value, default(T)))
  {
    // ……
  }
  // ……
}

或者

private bool IsDefault<T>(T value) where T : class
{
  if (value == null)
  {
    // ……
  }
  // ……
}
