Assembly.Load 的参数包括待加载的dll的完整规范。如果使用另一个方法，你可能会得到一个意料之外的dll。

此规则会在调用 Assembly.LoadFrom，Assembly.LoadFile 或 Assembly.LoadWithPartialName 时引发问题。

不合规的代码示例

static void Main(string[] args)
{
    Assembly.LoadFrom(...); // 不合规
    Assembly.LoadFile(...); // 不合规
    Assembly.LoadWithPartialName(...); // 不合规＋弃用的
}
