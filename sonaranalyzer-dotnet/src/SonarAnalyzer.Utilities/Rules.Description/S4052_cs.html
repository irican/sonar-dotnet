随着.NET 框架版本2的出现，某些实践已经过时。

特别是，异常现在应该扩展System.Exception，而不是System.ApplicationException。类似地，应该使用泛型集合，而不是旧的非泛型集合。当最终建立XML视图时，不应当继承 System.Xml.XmlDocument 类。

当外部可见的类型扩展了下列其中一种类型时，这条规则会引发一个问题:

  • System.ApplicationException
  • System.Xml.XmlDocument
  • System.Collections.CollectionBase
  • System.Collections.DictionaryBase
  • System.Collections.Queue
  • System.Collections.ReadOnlyCollectionBase
  • System.Collections.SortedList
  • System.Collections.Stack

不合规的代码示例

using System;
using System.Collections;

namespace MyLibrary
{
  public class MyCollection : CollectionBase  // 不合规
  {
  }
}

合规的解决方案

using System;
using System.Collections;

namespace MyLibrary
{
  public class MyCollection : Collection<T>
  {
  }
}
