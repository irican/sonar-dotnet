GetHashCode 用于将一个对象归档到一个 Dictionary 或 Hashtable中。如果 GetHashCode 操作的不是 readonly 类型的字段，这些字段如果在该对象存储后发生改变，该对象会在 Hashtable 中变成未归档的（mis-filed），后续的任何测试来检查该对象是否在 Hashtable 内时，都会返回假（false negative）。

不合规的代码示例

public class Person
{
  public int age;
  public string name;

  public override int GetHashCode()
  {
    int hash = 12;
    hash += this.age.GetHashCode(); // 不合规
    hash += this.name.GetHashCode(); // 不合规
    return hash;
  }

合规的解决方案

public class Person
{
  public readonly DateTime birthday;
  public string name;

  public override int GetHashCode()
  {
    int hash = 12;
    hash += this.birthday.GetHashCode();
    return hash;
  }
