最多只执行一次迭代的循环相当于使用了 if 语句，即有条件地执行一段代码。如果作者的初衷确实是有条件地执行一段代码，那么就应该改用 if；如果不是，循环的主体应该以更适当的方式使用内嵌的 return，break 或者throw 语句。

不合规的代码示例

for (int i = 0; i < 10; i++)
{
    Console.WriteLine(i);
    break;  // 不合规，循环只执行了一次
}
...
foreach (var item in items)
{
    return item;  // 不合规，循环只执行了一次
}
...

合规的解决方案

for (int i = 0; i < 10; i++)
{
    Console.WriteLine(i);
}
...
var item = items.FirstOrDefault();
if (item != null)
{
    return item;
}
...
