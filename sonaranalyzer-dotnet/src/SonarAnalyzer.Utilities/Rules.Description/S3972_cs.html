每条语句都占一行的代码是最清晰的。但是，将 if 条件与表示它的结果的语句写在一行也是一种常见的模式。然而把 if 与之前的 else 或 else if 语句的 } 写在同一行，要么是一个错误——丢失了 else 语句——要么可能会在将来造成错误，因为维护人员可能意识不到这两个语句不是连在一起的。

不合规的代码示例

if (condition1) {
  // ……
} if (condition2) {  // 不合规
  //……
}

合规的解决方案

if (condition1) {
  // ……
} else if (condition2) {
  //……
}

或者

if (condition1) {
  // ……
}

if (condition2) {
  //……
}
