enum 通常用于表示一组值中的不同元素。但是 enum 可以视为位字段，能对它们进行按位操作来组合它们的值。这是一种用单个数值就能指定多个元素的好方法。如果真的这样使用 enum，最好用 FlagsAttribute 标记 enum。

不合规的代码示例

enum Permissions
{
  None = 0,
  Read = 1,
  Write = 2,
  Execute = 4
}
// ……

var x = Permissions.Read | Permissions.Write;  // 不合规，没有使用[Flags]标记enum

合规的解决方案

[Flags]
enum Permissions
{
  None = 0,
  Read = 1,
  Write = 2,
  Execute = 4
}
// ……

var x = Permissions.Read | Permissions.Write;
