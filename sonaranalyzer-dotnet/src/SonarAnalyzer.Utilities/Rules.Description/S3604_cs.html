字段，属性和事件可以内联（inline）初始化或在构造函数中初始化。但同时使用这两种方法初始化则是多余的，内联初始化会被重写。

不合规的代码示例

class Person
{
  int age = 42; // 不合规
  public Person(int age)
  {
    this.age = age;
  }
}

合规的解决方案

class Person
{
  int age;
  public Person(int age)
  {
    this.age = age;
  }
}

例外

如果不是所有的构造函数都初始化某个字段则此规则不会引发问题。如果该字段被内联初始化为它的默认值，那么规则{rule:csharpsquid:S3052} 会报告此次初始化。
