静态字段的初始化是按照它们在类中出现的顺序，由上到下地执行的。因此如果一个静态字段的初始化需要用到其他的字段，而它在类中的位置又在这些字段之前，就很可能出现意外的结果。

不合规的代码示例

class MyClass
{
  public static int X = Y; // 不合规，此时Y的值是默认值，即0
  public static int Y = 42;
}

合规的解决方案

class MyClass
{
  public static int Y = 42;
  public static int X = Y;
}

或者

class MyClass
{
  public static int X;
  public static int Y = 42;

  static MyClass()
  {
    X = Y;
  }
}
