使用VarArgs调用约定的方法不符合公共语言规范(Common Language Specification, CLS)，而且可能无法跨编程语言访问，而params关键字以相同的方式工作，并且符合CLS。

当public或protected类型包含使用VarArgs调用约定的public或protected方法时，该规则会引发一个问题。

不合规的代码示例

using System;

namespace MyLibrary
{
    public class Foo
    {
        public void Bar(__arglist) // 不合规
        {
            ArgIterator argumentIterator = new ArgIterator(__arglist);
            for(int i = 0; i < argumentIterator.GetRemainingCount(); i++)
            {
                Console.WriteLine(
                    __refvalue(argumentIterator.GetNextArg(), string));
            }
        }
    }
}

合规的解决方案

using System;

[assembly: CLSCompliant(true)]
namespace MyLibrary
{
    public class Foo
    {
        public void Bar(params string[] wordList)
        {
            for(int i = 0; i < wordList.Length; i++)
            {
                Console.WriteLine(wordList[i]);
            }
        }
    }
}

例外

使用VarArgs调用约定的互操作方法不会引发问题。

[DllImport("msvcrt40.dll")]
public static extern int printf(string format, __arglist); // 合规的
