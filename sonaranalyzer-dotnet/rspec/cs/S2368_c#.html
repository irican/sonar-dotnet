使用多维数组做参数的公开方法需要开发人员掌握语言的高级知识才能使用。此外，传递给此类参数的内容并不直观，因此这样的方法不应对外公开，但在内部可以使用。

不合规的代码示例

public class Program
{
    public void WriteMatrix(int[][] matrix) // 不合规
    {
    }
}

合规的解决方案

public class Matrix
{
    // ……
}

public class Program
{
    public void WriteMatrix(Matrix matrix) // 合规
    {
    }
}
