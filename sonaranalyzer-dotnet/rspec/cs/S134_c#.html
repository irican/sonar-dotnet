嵌套的 if，switch，for，foreach，while，do，以及 try 语句会形成所谓的“Spaghetti code”。

这样的代码很难阅读、重构，也就很难去维护。

不合规的代码示例

默认的嵌套层数阈值为3时：

if (condition1) // 合规－depth = 1
{
  /* ... */
  if (condition2) // 合规－depth = 2
  {
    /* ... */
    for(int i = 0; i < 10; i++) // 合规－depth = 3, 未超过限制
    {
      /* ... */
      if (condition4) // 不合规－depth = 4
      {
        if (condition5) // Depth = 5，超过了限制，但只在depth = 4报错
        {
          /* ... */
        }
        return;
      }
    }
  }
}
