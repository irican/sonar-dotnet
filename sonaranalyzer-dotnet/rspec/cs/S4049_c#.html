属性像字段一样被访问，这使得它们更容易使用。

当public 或 protected 方法的名称以 Get开始，不接受任何参数，并返回一个不是数组的值时，该规则会引发一个问题。

不合规的代码示例

using System;

namespace MyLibrary
{
    public class Foo
    {
        private string name;

        public string GetName()  // 不合规
        {
            return name;
        }
    }
}

合规的解决方案

using System;

namespace MyLibrary
{
    public class Foo
    {
        private string name;

        public string Name
        {
            get
            {
                return name;
            }
        }
    }
}

例外

当方法出现以下问题时，该规则不会引发问题：

  • 是一个构造函数
  • 是一个override
  • 是一个接口实现
  • 是async
  • 返回Task，Task<T>
  • 被命名为：GetEnumerator，GetAwaiter
