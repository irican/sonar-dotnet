某些字符，一旦规范化为小写，就不能反复。也就是说，它们不能从一种语言环境转换到另一种语言环境，然后精确地恢复到原来的字符。

因此，强烈建议将字符和字符串规范化为大写。

不合规的代码示例

Thread.CurrentThread.CurrentCulture = new CultureInfo("tr-TR");
var areStringEqual = "INTEGER".ToLower() == "integer"; // 不合规，结果是假ToLower将解决“ınteger”
var areCharEqual = char.ToLower('I') == 'i'; // 不合规，结果是假ToLower将解决“ı”

var incorrectRoundtrip = "İ".ToLowerInvariant().ToUpper() == "I".ToLowerInvariant().ToUpper(); // 不合规，因为值越低，我们就会丢失关于正确大写字母的信息

合规的解决方案

Thread.CurrentThread.CurrentCulture = new CultureInfo("tr-TR");
var areStringEqual = "ınteger".ToUpperInvariant() == "ıNTEGER";
var areCharEqual = char.ToUpperInvariant('ı') == 'ı';
var correctRoundtrip = "İ".ToUpperInvariant().ToLower() != "I".ToUpperInvariant().ToLower();
