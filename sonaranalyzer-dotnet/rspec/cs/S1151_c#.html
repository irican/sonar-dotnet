switch 语句应仅用于明确定义控制流中的一些新分支。一条 case 分句包含的语句太多会大大降低整个控制流语句的可读性，这时应该把该 case 分句下的所有内容总结到一个方法里。

不合规的代码示例

默认阈值为3条语句时：

switch (myVariable)
{
    case 0: // 不合规：分句中包含了5条语句
        methodCall1("");
        methodCall2("");
        methodCall3("");
        methodCall4("");
        break;
    case 1:
        ...
}

合规的解决方案

switch (myVariable)
{
    case 0:
        DoSomething()
        break;
    case 1:
        ...
}
...
private void DoSomething()
{
    methodCall1("");
    methodCall2("");
    methodCall3("");
    methodCall4("");
}
