The PartCreationPolicyAttribute 属性是托管扩展框架(MEF)的一部分，用于指定如何创建导出的对象。因此，不使用 ExportAttribute 属性导出具有此属性的类是没有意义的。

当一个类被标记为与一个 PartCreationPolicyAttribute 共享，但缺少一个ExportAttribute时，这个规则会引发一个问题。

不合规的代码示例

[PartCreationPolicy(CreationPolicy.Any)] // 不合规
public class FooBar : IFooBar
{
}

合规的解决方案

[Export(typeof(IFooBar))]
[PartCreationPolicy(CreationPolicy.Any)]
public class FooBar : IFooBar
{
}
