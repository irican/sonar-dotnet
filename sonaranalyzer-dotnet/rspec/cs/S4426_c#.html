在生成密码学密钥(或密钥对)时，重要的是使用密钥长度提供足够的熵来抵御暴力攻击。对于RSA算法，密钥至少应该是2048位长。

当RSA密钥对生成器初始化时长度参数太小时，此规则会引发问题。

不合规的代码示例

using System;
using System.Security.Cryptography;

namespace MyLibrary
{
    public class MyCryptoClass
    {
        static void Main()
        {
            RSACryptoServiceProvider RSA = new RSACryptoServiceProvider(1024); // 不合规
            // ...
        }
    }
}

合规的解决方案

using System;
using System.Security.Cryptography;

namespace MyLibrary
{
    public class MyCryptoClass
    {
        static void Main()
        {
            RSACryptoServiceProvider RSA = new RSACryptoServiceProvider(2048);
            // ...
        }
    }
}
