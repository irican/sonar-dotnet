SafeHandle.DangerousGetHandle 方法很危险，这一点并不奇怪。因为该方法可能无法返回有效的句柄，使用它可能会产生泄露和漏洞。成功调用此方法并不困难，但正确地使用它则非常非常困难。因此应该直接避免使用它。

不合规的代码示例

static void Main(string[] args)
{
    System.Reflection.FieldInfo fieldInfo = ...;
    SafeHandle handle = (SafeHandle)fieldInfo.GetValue(rKey);
    IntPtr dangerousHandle = handle.DangerousGetHandle();  // 不合规
}
