只存储属性值的私有字段是一个典型的死亡存储（dead store）的例子。改变其属性值不会奏效，也意味着这里的代码多半存在严重错误。

不合规的代码示例

public class Rectangle
{
  private readonly int length;
  private readonly int width;  // 宽度数据被写过，但是没有读过

  public Rectangle(int length, int width)
  {
    this.length = length;
    this.width = width;
  }

  public int Surface
  {
    get
    {
      return length * length;
    }
  }
}

合规的解决方案

public class Rectangle
{
  private readonly int length;
  private readonly int width;

  public Rectangle(int length, int width)
  {
    this.length = length;
    this.width= width;
  }

  public int Surface
  {
    get
    {
      return length * width;
    }
  }
}
