有些操作尽管你 能 做，但并不意味着你就应该这么做，嵌套的三元运算就是这样。使用嵌套的三元运算符写出的代码，也许在你写的时候觉得意思很明确，但在六个月后维护人员（更糟的是也有可能是你自己）看到它们只能挠头并诅咒。

所以相反地，应该力求代码的意图明确。此时应另起一行，将嵌套的三元运算符分开写作独立的语句。

不合规的代码示例

public string GetTitle(Person p)
{
  return p.Gender == Gender.MALE ? "Mr. " : p.IsMarried ? "Mrs. " : "Miss ";  // 不合规
}

合规的解决方案

public string GetTitle(Person p)
{
  if (p.Gender == Gender.MALE)
  {
    return "Mr. ";
  }
  return p.IsMarried ? "Mrs. " : "Miss ";
}
