方法的主体（method body）为空可能是由于以下原因：

  • 无心的省略，之后应该将其补全。
  • 该方法尚不受支持或永远不受支持。这种情况下应抛出一个 NotSupportedException 异常。
  • 该方法是有意留为空白的重写方法。这种情况下应有嵌套的注释以说明这样做的原因。

不合规的代码示例

public override void DoSomething()
{
}

public override void DoSomethingElse()
{
}

合规的解决方案

public override void DoSomething()
{
  // 此处由于某种原因没有操作。
}

public override void DoSomethingElse()
{
  throw new NotSupportedException();
}

例外

此规则忽略以下方法：

  • 空的 virtual 方法，
  • 重写了 abstract 方法的空方法，
  • 测试程序集（test assemblies）中空的重写方法。
