在有位域标记的枚举类型中统一使用“None”来表示清除所有的标志值（flag values）。因此0值不应再表示任何其他的状态，否则会检查不到已经设置了bit 0。

不合规的代码示例

[Flags]
enum FruitType
{
    Void = 0,        // 不合规
    Banana = 1,
    Orange = 2,
    Strawberry = 4
}

合规的解决方案

[Flags]
enum FruitType
{
    None = 0,        // 合规
    Banana = 1,
    Orange = 2,
    Strawberry = 4
}
