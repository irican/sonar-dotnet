每个源文件都应该应该以一个标头（header）开始，说明文件的所有权和分发应用程序时必须使用的证书。

此规则必须与每个文件开头预期的标头文本一起提供。

如果你想让文件标头和你的代码首行（using，namespace……）用空行隔开的话，headerFormat 必须以空行结尾。

例如，如果你想让源文件呈以下格式

// Copyright (c) SonarSource. All Rights Reserved. Licensed under the LGPL License.  See License.txt in the project root for license information.

namespace Foo
{
}

那么 headerFormat 参数应该像这样以空行结尾

// Copyright (c) SonarSource. All Rights Reserved. Licensed under the LGPL License.  See License.txt in the project root for license information.
